# 🌿 Дневник состояния

Личный дневник настроения на React + Vite. Записи сохраняются в Supabase,
поэтому дневник можно открыть по одной ссылке с любого устройства. Перед
дневником показывается экран входа по PIN-коду.

## Настройка Supabase

1. Открой проект Supabase:
   `https://ipernfhcinbcnoczczmk.supabase.co`
2. В Supabase Dashboard перейди в **SQL Editor**.
3. Скопируй содержимое файла `supabase-schema.sql`.
4. Вставь SQL в редактор и нажми **Run**.
5. После этого запусти приложение локально или задеплой его.

В текущей настройке дневник защищён PIN-кодом на стороне приложения. Это удобная
защита от случайного доступа по ссылке, но не полноценная серверная авторизация.
Для более строгой защиты можно добавить Supabase Auth и Row Level Security по
пользователю.

## Запуск в WebStorm

1. Распакуй проект и открой папку в WebStorm.
2. В терминале (встроенном в WebStorm) выполни:
   ```bash
   npm install
   npm run dev
   ```
3. Открой ссылку из терминала (обычно `http://localhost:5173`).

Либо просто нажми на зелёный треугольник рядом со скриптом `dev` в
`package.json` — WebStorm сам всё запустит.

## Сборка для продакшена

```bash
npm run build
```

Готовые статические файлы появятся в папке `dist/` — их можно положить
на любой хостинг статики.

## Как открыть с планшета/другого компьютера

Чтобы пользоваться дневником с разных устройств, задеплой приложение один раз
и открывай его по ссылке:

- **GitHub Pages** — самый простой вариант, раз уже есть GitHub:
  1. Запушь проект в репозиторий.
  2. `npm run build`, затем задеплой содержимое `dist/` в ветку
     `gh-pages` (можно через пакет `gh-pages` или GitHub Actions).
  3. Открывай сайт по ссылке `https://<юзернейм>.github.io/<репо>/`
     с любого устройства.
- **Vercel / Netlify** — подключаешь репозиторий, деплой автоматический
  при каждом пуше, ничего вручную настраивать не нужно.

Внизу вкладки «История» есть кнопка **«Экспорт в JSON»** — ей можно выгрузить
все записи в файл и сохранить резервную копию.

## Структура проекта

```
src/
  App.jsx                — вкладки (новая запись / история / статистика)
  components/
    EntryForm.jsx        — форма записи (повторяет твою анкету)
    EntryList.jsx         — список записей в истории
    EntryDetail.jsx       — просмотр одной записи
    PinGate.jsx           — вход по PIN-коду
    ScaleSlider.jsx        — шкалы 0–5 / 0–10
    MultiSelect.jsx       — множественный выбор (эмоции, тело, что помогло)
    Stats.jsx             — график динамики (recharts)
  data/options.js         — списки эмоций/ощущений/вариантов (легко редактировать)
  lib/supabase.js         — подключение к Supabase
  utils/storage.js        — чтение/запись Supabase + экспорт в JSON
```

Чтобы добавить/убрать пункты анкеты (эмоции, ощущения в теле и т.д.) —
достаточно поправить массивы в `src/data/options.js`.
